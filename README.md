# Experiencia Web Interactiva: Amor de Telenovela

Este proyecto es una infografía narrativa digital que explora la idealización del romanticismo en las telenovelas latinoamericanas desde una perspectiva académica y psicosocial.

## Cómo visualizar

Simplemente abre el archivo `index.html` en cualquier navegador web moderno (Chrome, Edge, Firefox, Safari).

## Características Técnicas

- **Diseño Responsive**: Se adapta a móviles y escritorio.
- **Scroll Telling**: La narrativa avanza conforme el usuario hace scroll.
- **Motion Graphics**: Animaciones CSS y Parallax con Javascript.
- **Estética**: Diseño Editorial Digital con Glassmorphism.

## Estructura del Contenido

1. **Investigación Visual**: Introducción al concepto.
2. **Acto I**: La Promesa del Amor (Mito de la completitud).
3. **Acto II**: El Sacrificio (Roles de género).
4. **Acto III**: Normalización del Dolor (Melodrama).
5. **Acto IV**: Ficción vs. Realidad (Comparador interactivo).
6. **Acto V**: Despertar (Conclusión crítica).
